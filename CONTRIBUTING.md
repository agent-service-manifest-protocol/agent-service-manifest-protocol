# Contributing to ASMP

Thank you for your interest in the Agent Service Manifest Protocol.

## How to Contribute

### Spec Feedback
- Open an issue describing the problem or gap you see
- Propose changes via pull request to the spec

### Reference Implementation
- The reference registration server is in the `reference/` repo
- Bug fixes and improvements welcome via PR

### Examples
- Add `.asmp.yaml` examples for common services
- Share real-world usage patterns

### SDKs
- Python and TypeScript SDKs in separate repos
- Contributions to other language SDKs welcome

See [CODE_OF_CONDUCT.md](CODE_OF_CONDUCT.md).
